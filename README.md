# Pony

## Introduction

使用 Canvas 实现的简易停车场平面图，并使用中间层转发 MQTT 协议实现与阿里云 IoT 平台的通信。

## Installation

```bash
bun install
```

## Usage

> 参考`shell.nix`环境变量配置自建`.env`。

```bash
bun dev # or just run `nix develop .` with nix installed
```
